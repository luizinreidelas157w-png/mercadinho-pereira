Trabalho de Conclusão de Curso - Técnico em Informática

Sobre o Projeto

Este projeto apresenta o desenvolvimento de um sistema de reserva online para o Mercadinho Pereira, permitindo que clientes reservem produtos pela internet e retirem posteriormente na loja física.

Objetivo

Desenvolver uma plataforma simples e funcional que facilite o processo de reserva de produtos, modernizando o atendimento e melhorando a organização do estabelecimento.

Tecnologias Utilizadas

  • HTML
  • CSS
  • JavaScript
  • PHP
  • MySQL
